# Graph Pathfinder

Build with:
```
cd .\GraphPathfinder\
dotnet run
```

### Base Requirements
- .NET Desktop Runtime 9.0 installed
- 1280x960 screen resolution or higher
- Windows 10 22H2 (x86/x64/Arm64) or higher
- 2GB RAM
- 2-core CPU ≥ 1.4 GHz with SSE2 support
- GPU with DirectX 9 (WDDM 1.0) and ≥ 60MB VRAM
